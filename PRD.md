一、基础信息
角色来源：蔚蓝档案 - 中务桐乃
设计主题：网站配色参考，色彩灵感取自角色服装
整体风格定位：清新、科技、活力、专业
二、完整色板（含色值）

1. 主色调 Primary Colors（5 色）
   #7A6DF1 深紫主色
   #B9ADF7 浅紫
   #9EDDF6 浅青蓝
   #323859 深藏黑
   #F2F4F8 极浅灰白
2. 辅助色 Secondary Colors（4 色）
   #FFD68A 暖黄
   #FFB4CF 柔粉色
   #A8B7FF 淡蓝紫
   #E6F7FA 浅青白
3. 中性色 Neutral Colors（6 色，文本 / 背景用）
   #1A1F2E 纯黑（正文深色）
   #3B4252 深灰（次要文本）
   #5C6373 中灰（说明文字）
   #A0A6B3 浅灰（占位、禁用文字）
   #D8DEE9 淡灰（分割线、边框）
   #F5F7FA 纯白背景底色
   三、界面色彩占比分配
   #7A6DF1 深紫：30%（核心主视觉）
   #B9ADF7 浅紫：20%
   #9EDDF6 浅青蓝：15%
   #323859 深藏黑：15%
   #FFD68A 暖黄：5%
   #FFB4CF 柔粉：5%
   #A8B7FF 淡蓝紫：5%
   #E6F7FA 浅青白：5%
   四、UI 组件配色示例规范
4. 按钮 Buttons
   主按钮 Primary：填充 #7A6DF1 深紫
   次按钮 Secondary：白色底、#7A6DF1 紫色边框
   文本按钮 Text Link：紫色文字 + 右侧箭头
5. 导航栏 Navbar
   文字默认中性深灰，当前激活页面文字使用 #7A6DF1 深紫 + 底部紫色下划线
   左侧搭配紫色线性图标，右侧用户头像圆形控件
6. 标签 Tags
   活动标签：浅紫底色
   重要标签：深紫底色白色字
   更新标签：淡蓝底色
7. 输入框 Input
   常规搜索框：浅灰边框、浅灰占位文字
   激活输入框：#7A6DF1 紫色边框
   不可用状态：浅灰底色 + 浅灰文字
8. 卡片 Card
   白底卡片，搭配中性浅灰分割线；标题深灰正文、辅助文字中灰；进度条底色浅灰，填充 #7A6DF1 深紫；右下角放置主紫色行动按钮
9. 状态提示 Status
   完成：绿色勾选标识
   进行中：浅蓝空心圆点
   待处理：黄色空心圆圈
   已取消：灰色叉号标识
   五、渐变与视觉特效
   主渐变 Primary Gradient：#7A6DF1 → #B9ADF7（深紫过渡浅紫）
   辅助渐变 Secondary Gradient：#9EDDF6 → #A8B7FF（浅青蓝过渡淡蓝紫）
   光效 Glow Effect：主紫色圆形柔化发光效果
